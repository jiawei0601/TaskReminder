# 週期任務提醒工具 (Task Reminder Tool)

這是一個現代化、美觀的週期性任務提醒工具，幫助您建立良好的工作習慣。

## 功能特點
- **自定義週期**: 設定任意分鐘數進行提醒。
- **現代化介面**: 採用深色模式、磨砂玻璃質感 (Glassmorphism) 及動態背景。
- **即時倒數**: 每個任務都有獨立的倒數計時與進度條。
- **系統通知**: 當時間到達時，會發出瀏覽器系統通知。
- **完成紀錄**: 自動分類與存儲您的完成歷史。
- **數據持久化**: 使用 `localStorage` 將數據儲存在瀏覽器中。

## 技術棧
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Fonts (Inter, Noto Sans TC)

## 如何使用
1. 輸入您想要進行的任務名稱。
2. 設定提醒週期（以分鐘為單位）。
3. 點擊「新增任務」。
4. 當提醒視窗彈出時，點擊「完成並記錄」。
